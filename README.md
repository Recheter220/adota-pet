# Instalação do projeto

- Baixe e instale a última versão do [Herd](https://herd.laravel.com/download/latest/windows)
- Clone o projeto na sua máquina
- No menu *Sites* do Herd, clique no botão (+) Add, depois *link existing project*. Selecione a pasta em que clonou o projeto. Depois, confirme a versão 8.4 do PHP e avance até a conclusão.
- Abra a pasta instalada no seu terminal e execute os seguintes comandos:
```
composer install
npm install
```

# Executando o projeto
- Pelo gerenciador de sites do Herd, clique no botão Open na linha VS Code
- No terminal do VS Code, digite o comando:
```
npm run dev
```
- Pronto! O projeto já está rodando e atualizando automaticamente com as suas alterações!
